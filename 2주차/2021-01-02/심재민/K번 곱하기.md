## [입력]
- 자연수 N과 K를 입력합니다.
```math
(1 <= N <= 1,000,000,000)
(1 <= K <= 50)
(1 <= K <= 50)
```
## [출력]
```math
1^K + 2^K + ... + N^K 를 1,000,000,009로 나눈 나머지를 출력합니다.
```
## [풀이]
- 아직 해결하지 못함 (60점)
- 시간 초과 오류 (테스트 4,5)
- 반복문으로 코드를 작성했을 때 , N이 커지는 만큼 시간도 오래 걸리기 때문에 시간 초과 오류가 발생하는 것으로 보임.
- 스터디 시간에 더 좋은 풀이가 있는지 튜터님께 `질문`
